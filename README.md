# Exemples-ESP32 - A05 Variables

## Descripción
Ejemplos prácticos de uso de variables en ESP32 con Arduino Core.

## Tipos de Variables Implementados
- `const` - Constantes (valores que no cambian)
- `int` - Números enteros
- `long` - Números enteros grandes  
- `float` - Números con decimales
- `bool` - Valores booleanos (true/false)
- `char` - Caracteres individuales

## Uso en Wokwi
1. Abrir proyecto en Wokwi.com
2. Compilar y ejecutar
3. Abrir Monitor Serie a 115200 baudios
